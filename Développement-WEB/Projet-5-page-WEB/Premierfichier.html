4 - Ajouter des éléments multimédia

Pour insérer une image, on utilise la balise <img>. L'attribut src spécifie le chemin vers l'image, et l'attribut alt fournit une description alternative pour l'image (utile pour l'accessibilité et si l'image ne peut pas être affichée)

Exemple 1: <img src="images/logo.png" alt="Logo de l'entreprise" >

Exemple 2: <img src="images/paysage.png" alt="Photo de paysage">

Différencier chemin relatif et chemin absolu :

Chemin relatif :
Le chemin est relatif à la page web actuelle. Par exemple, si l'image est dans un dossier "images" à côté de la page web, le chemin relatif sera "images/nom_image.jpg".
Chemin absolu :
Le chemin complet depuis la racine du site web ou même depuis un autre site web. Par exemple : "/images/nom_image.jpg" (depuis la racine du site) ou "https://www.example.com/images/nom_image.jpg"

4.1 Spécifier la taille des images

On utilise les attributs width et height pour définir la largeur et la hauteur de l'image en pixels

4.2 Exemple :

taille originale
<img src="images/paysage.png" alt="Paysage">
taille personnalisée
<img src="images/paysage.png" alt="Paysage" width="200" height="125">
<img src="images/paysage.png" alt="Paysage" width="400" height="250">

Les formats courants sont :
JPEG : Idéal pour les photographies avec des dégradés de couleur.
PNG : Supporte la transparence et est idéal pour les graphiques.
GIF : Pour les animations simples.
SVG : Graphiques vectoriels, parfaits pour les logos et icônes, car ils se redimensionnent sans perte de qualité

Insérer des vidéos :

On utilise la balise < video > pour intégrer des vidéos. Les attributs src, width, height et controls (pour afficher les contrôles de lecture) sont importants

Exemple :
